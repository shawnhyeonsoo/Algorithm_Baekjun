문제 </br>
</br>두 정수 A와 B를 입력받은 다음, A+B를 출력하는 프로그램을 작성하시오. </br>
</br>
입력</br>
첫째 줄에 A와 B가 주어진다. (0 < A, B < 10) </br>

</br>출력</br>
첫째 줄에 A+B를 출력한다.</br>

</br>
</br>
솔루션:</br>

```python
a, b = map(int,input().split())
print(a+b)
```
